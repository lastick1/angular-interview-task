# Постановка задачи

От бизнеса поступило требование сделать прототип/mvp формы добавления услуги. На предыдущем исполнителе задачи сработал bus factor, реализацию необходимо закончить

4 поля:
1. Выбор из списка. Поставщик услуги (provider). Возможные значения: партнёр, банк.
2. Выбор из списка. Наименование услуги (name). Справочник необходимо отфильтровать из справочника услуг по полю "Поставщик услуги".
3. Число. Срок. Условное отображение: только если услуга от партнёра.
4. Число. Цена услуги (price). Если "Поставщик услуги" = партнёр, то значение только для чтения и заполняется из поля price соответствующей услуги.

```typescript
const SERVICES = [
  { provider: 'bank', name: 'Кредит' },
  { provider: 'bank', name: 'Лизинг' },
  { provider: 'partner', name: 'Страхование', price: 776 },
  { provider: 'partner', name: 'Консалтинг', price: 72 },
];
```
